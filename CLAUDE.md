# skills

Agent skills repo. One skill per folder in `skills/<name>/`, each with a `SKILL.md` and `agents/openai.yaml`.

## Rules

- Everything in English: commits, code, docs.
- Ultra-concise answers. No preamble, no recap, no rejected options.
- Commits: conventional commits, one line, never `Co-Authored-By`.
- File names in kebab-case.
